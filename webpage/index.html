üí° What is the benefit of a manual key exchange?

Manual key exchange ‚Äî like scanning a QR code in person or verifying a code over a call ‚Äî brings humans back into the loop. It allows people to:

  • Personally verify identity (face, voice, presence)
  • Start a secure relationship without relying on central servers

DeepTrust builds a cryptographic layer on top of this gold standard to extend that trust into the digital world.

üîê Why does DeepTrust use two separate sets of keys for every relationship?

Each relationship has two asymmetric key pairs ‚Äî one per direction. This means:

  • You can revoke or rotate trust from one direction without breaking the other
  • Each party controls what they share and with whom
  • Authentication is directional, not assumed to be mutual
üïí Why does DeepTrust use two TOTP secrets per relationship?

Each user generates a unique TOTP secret for the other person. This allows:

  • Independent verification by each party
  • Explicit mutual authentication ‚Äî never assumed
  • Resilience: if one secret is compromised, the other remains secure
